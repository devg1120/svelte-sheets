# 对齐

## Column

| 参数          | 说明     | 类型                            | 默认值 |
| ------------- | -------- | ------------------------------- | ------ |
| align         | 水平对齐 | `"left"`, `"center"`, `"right"` | center |
| verticalAlign | 垂直对齐 | `"top", "middle", "bottom"`     | middle |

## 水平

- 默认 `"center"`
- align 可为`"left"`, `"center"`, `"right"`

:::

## 垂直

- 默认 `"middle"`
- verticalAlign 可为`"top", "middle", "bottom"`
