# 固定

## Column

| 参数  | 说明       | 类型                | 默认值 |
| ----- | ---------- | ------------------- | ------ |
| fixed | 是否固定列 | `"left"`, `"right"` | 100    |


## 固定表头

- `config.HEIGHT`高度，设置高度表示固定表头

::: demo

fixed/fixed-header
h:400px
h:350px
:::

## 不固定表头

- `config.MAX_HEIGHT` 和`config.HEIGHT`为0，表示不固定表头

::: demo

fixed/no-fixed-header
h:400px
:::

## 固定列

- fixed 可为`"left","right"`

::: demo

fixed/left-right
h:320px
:::
